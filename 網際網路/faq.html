如何連接網際網路？網際網路服務供應商扮演什麼角色？

您必須先與網際網路服務供應商 (ISP) 簽訂一份方案，才能開始探索網路世界。網際網路服務供應商是一家公司，可為您提供網際網路的存取權與其他網路服務。他們會提供不同的網路連線方式，包括透過撥號、纜線、光纖或 Wi-Fi。不同的連線方式決定了上網速度的快慢。


如何透過手機上網？與透過桌上型電腦上網不一樣嗎？

一般來說，透過手機上網以及撥打電話時，所使用的都是相同的無線訊號。手機是先連接到附近的基地台，再由基地台連接到網際網路。由於行動裝置與網際網路之間的數據傳輸成本相當高，所以服務供應商會按照數據方案進行收費。

像 Android 行動裝置這類裝置也可以透過 Wi-Fi 連接網際網路。Wi-Fi 可以讓您的手機、平板電腦或筆記型電腦透過無線訊號連接網際網路，不需要透過手機訊號或數據方案。一般來說，透過 Wi-Fi 連線時，手機的上網速度較快，不過前提是您所在地區必須能使用 Wi-Fi。許多咖啡廳、零售店與某些城市都會免費提供 Wi-Fi 連線。



什麼是網址、IP 位址和 DNS？它們又有什麼重要性？

網址是指您為了連接某個網站，而在瀏覽器中輸入的網路位址。每個網站都有一個網址，例如 www.google.com 這個網址可帶您前往 Google 的網站。

每個網址也有一個 IP 位址。IP 位址是一連串的號碼，能讓您的電腦知道到哪裡可以找到您想要的資料。IP 位址就像手機號碼一樣，只不過這一串號碼又長又複雜。由於 IP 位址極為複雜又不好記，於是網址就應運而生。如要前往 Google 網站，您不必輸入 IP 位址 (45.732.34.353)，只要輸入網址 www.google.com 即可。

網際網路上的網站和 IP 位址為數眾多，因此瀏覽器不可能知道每個網站或 IP 位址所在何處，必須逐一搜尋。這時 DNS (網域名稱系統) 就派上用場了。

DNS 基本上就是一本網路電話簿。只是在電話簿中「張三」會轉譯成電話號碼，而 DNS 則是將網址 www.google.com 轉譯成 IP 位址，然後帶您前往所需網站。



什麼是瀏覽器？

就像到圖書館「翻閱」群書一樣，您可以利用網路瀏覽器在網際網路上尋找或瀏覽網頁。瀏覽器是電腦上的一種軟體，可以讓您存取網際網路。瀏覽器就是一個視窗，當中為您顯示了資訊所在的各種網站。您唯一要做的就是在瀏覽器中輸入網址，然後就能立即前往該網址所指向的網站。


什麼是瀏覽器？

顧名思義，「搜尋」就是指尋找資料。舉例來說，如果在 Google 的搜尋引擎中輸入「汽車」這兩個字，您的裝置就會透過網際網路將搜尋要求傳送給 Google。Google 會尋找正確的搜尋結果，然後傳回到您的裝置，整個過程不用 1 秒鐘的時間。

為了找到結果，搜尋引擎會查閱及檢索網路上的所有資料。全世界的資訊量每兩年就增加一倍，要協助人們找到所需資訊日益困難，尤其在使用者每天執行的搜尋中，有 16% 是屬於全新的查詢內容。因此工程師必須不斷努力提升搜尋功能，讓企業和客戶可以持續各取所需。
