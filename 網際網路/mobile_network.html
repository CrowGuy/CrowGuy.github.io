行動網路

G的定義

術語中的「G=  generation 」代表的是安裝在手機和支持蜂窩網絡的移動通訊。每一代「G」都意味着你需要一部新的手機以及對網絡進行昂貴的升級。

「G」每更新一代都會帶來基礎技術的革命。越來越多的網絡都擁有為每一位用戶傳輸更多的數據的能力，與此同時，更提高通話質量的潛力。

通訊的進步

最早的兩代「G」分別稱為「模擬制式手機（1G）」和「第二代數字手機（2G）」。然而隨着技術的發展，「G」就變得複雜了。

第三代移動通訊技術，或者說3G技術是在2003年由美國首創的。在保證了最低144 Kb/s的持續性網速條件下，3G給我們帶來了移動寬帶。雖然現在有了各種各樣的3G技術，但是其中任何一個都已經能在任何位置達到400 Kb/s甚至是更高的網速。

4G的介紹

除了與現有網路的可兼容性外，4G要有更高的數據吞吐量、更低時延、更低的建設和運行維護成本、更高的鑒權能力和安全能力、支持多種QoS等級。從融和的角度看，4G意味著更多的參與方，更多技術、行業、應用的融合，不再局限於電信行業，或許區域網、網際網路、電信網、廣播網、衛星網等能夠融為一體組成一個通播網，無論使用什麼終端，都可以享受高品質的資訊服務，向寬頻無線化和無線寬頻化演進，使4G滲透到生活的方方面面。

4G的優點

  1. 通信速度快

    從移動通信系統數據傳輸速率作比較，第一代模擬式僅提供語音服務。第二代數位式移動通信系統傳輸速率也只有9.6Kbps，最高可達32Kbps，如PHS。第三代移動通信系統數據傳輸速率可達到2Mbps。第四代最高可以達到高達1000Mbps速度傳輸無線信息。

  2. 通信靈活

    4G通信使人們不僅可以隨時隨地通信，更可以雙向下載傳遞資料、圖畫、影像，當然更可以和從未謀面的陌生人網上連線對打遊戲。

  3. 智能性能高

    第四代移動通信的智能性更高，不僅表現于4G通信的終端設備的設計和操作具有智能化，例如對菜單和滾動操作的依賴程度會大大降低，更重要的4G手機可以實現許多難以想象的功能。

  4. 提供增值服務

    4G通信并不是從3G通信的基礎上經過簡單的升級而演變過來的，3G移動通信系統主要是以CDMA為核心技術，而4G移動通信系統技術則以正交多任務分頻技術(OFDM)最受矚目，；不過考慮到與3G通信的過渡性，第四代移動通信未來以OFDM為核心技術的第四代移動通信系統，也會結合兩項技術的優點，一部分會是以CDMA的延伸技術。

  5. 高質量通信

    儘管第三代移動通信系統也能實現各種多媒體通信，為此未來的第四代移動通信系統也稱為“多媒體移動通信。 第四代移動通信不僅僅是為了因應用戶數的增加，更重要的是，必須要因應多媒體的傳輸需求使可以容納市場龐大的用戶數、改善現有通信品質不良，以及達到高速數據傳輸的要求。

4G的缺點

  1. 容量受限

    人們對未來的4G通信的印象最深的莫過于它的通信傳輸速度會得到極大提升，但手機的速度會受到通信系統容量的限制，如系統容量有限，手機用戶越多，速度就越慢。所以4G手機會很難達到其理論速度。

  2. 技術難

    如何保證樓區、山區，及其它有障礙物等易受影響地區的信號強度等問題。另外在移交方面存在的技術問題，使手機很容易在從一個基站的覆蓋區域進入另一個基站的覆蓋區域時和網絡失去聯系。

  3. 設施更新慢

    在部署4G通信網絡系統之前，覆蓋全球的大部分無線基礎設施都是基于第三代移動通信系統建立的，如果要向第四代通信技術轉移的話，全球的許多無線基礎設施都需要經歷著大量的變化和更新，勢必減緩4G通信技術全面進入市場、占領市場的速度。

  4. 其他

    因為手機的功能越來越強大，而無線通信網絡也變得越來越復雜，同樣4G通信在功能日益增多的同時，它的建設和開發也會遇到比以前系統建設更多的困難和麻煩。
